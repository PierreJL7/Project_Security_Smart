¿Cómo puedo usar Alexa?

Para usar Alexa, necesitas un dispositivo habilitado para Alexa, como un altavoz Echo de Amazon. Simplemente di "Alexa" seguido de tu comando o pregunta. Puedes pedirle que reproduzca música, controle dispositivos domésticos inteligentes, proporcione información del tiempo, configure recordatorios y mucho más. Es como tener tu propio asistente personal en casa.
